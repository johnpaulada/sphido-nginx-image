# Markdown support

Sphido use [Parsedown](http://parsedown.org/)

## Mix Latte and Markdown

Use default `{l}url 'page'{r}` to get {url 'page'}

Try `{l}= 'Ahoj'{r}` to get {= 'Ahoj'}
